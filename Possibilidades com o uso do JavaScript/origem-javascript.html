Quando surgiu o JavaScript?

O JavaScript surgiu em 1995 e foi criado em apenas 10 dias por Brendan Eich, engenheiro da Netscape Communications. A linguagem foi inicialmente chamada de "Mocha", depois "LiveScript" e, finalmente, "JavaScript". O lançamento ocorreu durante o auge da competição entre navegadores, onde a Netscape e a Microsoft buscavam liderar o mercado.

A necessidade de uma linguagem de script que pudesse rodar no navegador surgiu para permitir que os sites fossem mais dinâmicos e interativos, além de oferecer uma alternativa ao back-end para manipulação de dados e validação de formulários diretamente no front-end.

Desde então, JavaScript passou por diversas evoluções e padronizações, como o ECMAScript (ECMA), que ajuda a manter uma versão unificada da linguagem em diferentes navegadores.
